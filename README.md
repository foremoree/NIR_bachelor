# НИР: "Исследование методов реконструкции 3D форм объектов по облаку точек"


Задача: реконструкции 3D форм объектов по облаку точек, получаемому техническими системами сканирования.
        (возможно несколько раз скнировали) 
        (англ.: Point Cloud => MESH)


Основные этапы выполнения НИРС:
1. Этап 1. Обзор алгоритмов (нейросетевых) и баз данных (бенчмарков) для решения задачи.    
2. Этап 2. Создание БД (собственной) изображений/видео для решения задачи, включающей:
           а) данные сканирования технической системов;
           б) данные программы моделирования облака точек по CAD-моделям деталей (разработать программу!).
           Нахождение, скачивание и анализ содержимого существующих БД для решения задачи...
3. Этап 3. Нахождение и апробация существующих алгоритмов (нейросетевых) решения задачи.    
       Тестирование их для преднастроенных параметров. Отбор лучших моделей. 
           Проведение вычислительных экспериментов, анализ результатов.
4. Этап 4. Обучение (файнтюнинг) отобранных лучших моделей, построение окончательного решения. 
           Проведение вычислительных экспериментов, анализ результатов.
